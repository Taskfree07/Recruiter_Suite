# Salary Dataset Directory

Place your Kaggle salary dataset CSV file here as `salary_data.csv`

## Instructions:

1. Download salary dataset from Kaggle (see SALARY_DATASET_SETUP.md)
2. Rename to `salary_data.csv`
3. Place in this directory
4. Run the app - model will train automatically!

## Example datasets:
- Data Science Salaries 2023
- Tech Salaries 2023
- US Salary Data

See `../SALARY_DATASET_SETUP.md` for full instructions.
